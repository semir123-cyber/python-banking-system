# Python Banking System (Flask Web App)

A simple banking web application built with Flask.

## Features
- User Registration & Login
- Deposit & Withdraw
- Transfer Money
- Transaction History (with timestamps)
- Clean UI Dashboard

## Tech Stack
- Python
- Flask
- HTML/CSS
- JSON (data storage)

## Future Improvements
- Database integration (SQLite/PostgreSQL)
- Authentication security upgrade
- Deployment online
